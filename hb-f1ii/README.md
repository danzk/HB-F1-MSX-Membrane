# MSX HB-F1II Keyboard Membrane

A replacement keyboard membrane for the Sony HB-F1II MSX home computer.

This variant reworks the HB-F1 PCB and uses the revised `Switch 2` footprint
from the shared footprint library.

See the [repository README](../README.md) for tooling and licence information.
